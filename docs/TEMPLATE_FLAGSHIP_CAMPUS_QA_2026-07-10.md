# 应届校招项目实习优先模板验收记录

日期：2026-07-10
对应路线图：M1「六套旗舰模板」中的第二套

## 产品目标

面向有课程项目、竞赛、实习或校园组织经验，但正式工作经历较少的应届生。模板以可证明的实践为正文主线，避免把大量空间浪费在空泛的自我评价和装饰元素上。

## 本轮交付

- 新增 `qm-student-editorial` 独立预览结构，不再复用通用模板 DOM。
- 教育背景压缩为顶部摘要，突出学校、专业学历、时间、GPA 与核心课程。
- 正文固定优先级：代表项目、实习经历、校园经历、技能工具、荣誉证书、个人优势。
- 求职意向合并到姓名区，减少重复模块占用的纵向空间。
- 新建校招简历时自动显示项目、实习、校园和荣誉模块，默认隐藏空的正式工作经验。
- 用户已有简历切换模板时仍以原简历内容和显示状态为准，不强制隐藏已有工作经历。
- 无真实照片时不渲染破损占位图；上传照片后仍支持右上角证件照。
- 网页预览与 PDF 打印样式同步。
- 模板库示例改为完整的应届生内容，包含教育、项目、实习、校园、技能、荣誉和个人优势。
- 更新 Mock 数据、模板元数据与后端种子数据，删除虚构使用量。
