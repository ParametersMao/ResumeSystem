# 技术开发 ATS 单栏模板验收记录

日期：2026-07-10
对应路线图：M1「六套旗舰模板」中的第一套

## 产品目标

面向前端、后端、测试和数据岗位，提供一套以机器解析和招聘者快速扫描为优先级的中文单栏模板。模板不依赖头像、图标、侧栏或视觉进度条表达关键信息。

## 本轮交付

- 新增独立的 `qm-minimal-ats` 预览结构，不再复用通用模板 DOM。
- 固定默认阅读顺序：个人总结、专业技能、工作经历、实习经历、项目经历、教育背景、荣誉证书、校园经历、求职意向、自定义模块、兴趣爱好。
- 工作和项目条目采用「岗位/角色优先、组织次之、时间右对齐」的信息层级。
- 技能使用纯文本分隔，不使用星级、百分比或图形进度条。
- 禁用头像，并同步网页预览与 PDF 打印样式。
- 更新模板名称、标签、推荐权重和种子数据，删除虚构使用量。
- 补齐本地 Mock 的模板详情、收藏和用户态链路，支持无后端的完整页面联调。
