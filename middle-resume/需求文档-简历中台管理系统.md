## 一、系统概述

### 1.1 产品定位

简历中台管理系统是一个面向平台运营人员和管理员的后台管理系统，提供：
- **用户管理**：C端用户和管理员管理
- **模板管理**：简历模板的CRUD和审核
- **数据统计**：用户行为、模板使用等数据分析
- **AI监控**：AI操作记录和用量监控
- **系统配置**：平台参数和权限配置

### 1.2 目标用户

| 用户角色 | 权限说明 | 主要职责 |
|---------|---------|---------|
| 超级管理员 (admin) | 所有权限 | 系统配置、用户管理、数据监控 |
| 运营人员 (operator) | 部分管理权限 | 模板管理、内容审核、数据分析 |
| 查看者 (viewer) | 只读权限 | 数据查看、报表导出 |

### 1.3 核心价值

1. **提升运营效率** - 统一管理后台，减少操作成本
2. **数据驱动决策** - 全面的数据统计和分析
3. **保障平台质量** - 模板审核和用户管理
4. **监控AI成本** - AI使用量监控和成本控制

## 二、功能需求

### 2.1 认证与权限

#### 2.1.1 管理员登录
**功能描述**: 管理员通过账号密码登录系统

**登录方式**:
- 用户名+密码
- 支持记住密码（本地加密存储）
- 支持验证码（防暴力破解）

**安全措施**:
- 密码MD5/BCrypt加密
- 连续5次失败锁定30分钟
- Token有效期24小时
- 支持强制下线

**接口定义**:
```typescript
POST /api/auth/login
Request: {
  username: string,
  password: string,
  captcha?: string,
  remember?: boolean
}
Response: {
  code: number,
  message: string,
  data: {
    token: string,
    user: {
      id: number,
      username: string,
      userType: 'admin' | 'operator' | 'viewer',
      permissions: string[]
    }
  }
}
```

#### 2.1.2 权限控制
**功能描述**: 基于角色的访问控制(RBAC)

**权限列表**:
```typescript
const permissions = {
  // 用户管理
  'user:view': '查看用户',
  'user:create': '创建用户',
  'user:edit': '编辑用户',
  'user:delete': '删除用户',
  'user:status': '启用/禁用用户',
  
  // 模板管理
  'template:view': '查看模板',
  'template:create': '创建模板',
  'template:edit': '编辑模板',
  'template:delete': '删除模板',
  'template:audit': '审核模板',
  
  // 数据统计
  'stats:view': '查看统计',
  'stats:export': '导出数据',
  
  // AI操作
  'ai:view': '查看AI记录',
  'ai:delete': '删除AI记录',
  
  // 系统配置
  'system:config': '系统配置',
  'system:admin': '管理员管理'
}
```

**角色权限矩阵**:
| 权限 | admin | operator | viewer |
|-----|-------|----------|--------|
| user:* | ✓ | ✓ (限制) | ✗ |
| template:* | ✓ | ✓ | ✗ |
| stats:* | ✓ | ✓ | ✓ |
| ai:* | ✓ | ✓ | ✓ (仅view) |
| system:* | ✓ | ✗ | ✗ |

**实现方式**:
- 前端：路由守卫+按钮级权限控制
- 后端：接口级权限验证

### 2.2 数据统计仪表盘

#### 2.2.1 数据概览
**功能描述**: 展示平台核心数据的总览

**展示指标**:
```typescript
interface DashboardStats {
  // 今日数据

    newUsers: number,          // 新增用户

    resumeExported: number,    // 导出简历数
    aiOperations: number       // AI操作次数

  total: {

    resumes: number,           // 总简历数
    templates: number,         // 总模板数
    aiTokenUsed: number        // AI总消耗Token
  },
  
  // 环比数据
  growth: {
    userGrowth: string,        // 用户增长率 "+12.5%"
    resumeGrowth: string,      // 简历增长率
    exportGrowth: string,      // 导出增长率
    aiGrowth: string          // AI使用增长率
  }
}
```

**数据卡片**:
- 卡片式展示
- 图标+数字+增长趋势
- 点击查看详情

**接口定义**:
```typescript
GET /api/statistics/dashboard
Response: {
  code: number,
  message: string,
  data: DashboardStats
}
```

#### 2.2.2 趋势图表
**功能描述**: 展示数据的时间趋势

**图表类型**:
1. **用户趋势图**（折线图）
   - X轴：日期
   - Y轴：用户数
   - 系列：新增用户、活跃用户
   - 时间范围：近7天/近30天/近90天

2. **简历操作趋势图**（折线图）
   - 系列：创建、编辑、导出
   - 支持按模板筛选

3. **AI使用趋势图**（柱状图）
   - 系列：生成、优化、建议
   - Y轴：操作次数

4. **模板使用排行榜**（横向柱状图）
   - Top 10模板
   - 显示使用次数和占比

**交互功能**:
- 时间范围选择
- 图例切换
- 数据点提示
- 图表导出（PNG/SVG）

**接口定义**:
```typescript
GET /api/statistics/trend
Query: {
  type: 'user' | 'resume' | 'ai' | 'template',
  startDate: string,
  endDate: string,
  granularity: 'day' | 'week' | 'month'
}
Response: {
  code: number,
  message: string,
  data: {
    labels: string[],      // X轴标签
    datasets: Array<{
      name: string,
      data: number[]
    }>
  }
}
```

#### 2.2.3 用户分析
**功能描述**: 用户行为和特征分析

**分析维度**:
1. **用户活跃度**
   - DAU、WAU、MAU
   - 留存率（次日/7日/30日）
   - 活跃时段分布

2. **用户画像**
   - 地域分布（地图）
   - 年龄分布（饼图）
   - 职业分布（饼图）

3. **用户行为**
   - 平均编辑时长
   - 平均导出次数
   - 功能使用偏好

**可视化组件**:
- 地图（基于ECharts）
- 饼图
- 热力图
- 漏斗图

#### 2.2.4 数据导出
**功能描述**: 导出统计数据为Excel/CSV

**导出内容**:
- 用户列表
- 简历列表
- AI操作记录
- 统计报表

**导出格式**:
- Excel (.xlsx)
- CSV (.csv)
- JSON (.json)

**接口定义**:
```typescript
GET /api/statistics/export
Query: {
  type: 'users' | 'resumes' | 'ai' | 'report',
  format: 'xlsx' | 'csv' | 'json',
  startDate?: string,
  endDate?: string
}
Response: {
  code: number,
  message: string,
  data: {
    url: string,           // 下载链接
    expireAt: string      // 过期时间
  }
}
```

**筛选条件**:

#### 2.3.2 用户详情

#### 4.3.1 权限控制模块
```typescript
// hooks/usePermission.ts
export function usePermission() {
  const authStore = useAuthStore()
  
  function hasPermission(permission: string): boolean {
    return authStore.permissions.includes(permission)
  }
  
  function hasAnyPermission(permissions: string[]): boolean {
    return permissions.some(p => hasPermission(p))
  }
  
  function hasAllPermissions(permissions: string[]): boolean {
    return permissions.every(p => hasPermission(p))
  }
  
  return {
    hasPermission,
    hasAnyPermission,
    hasAllPermissions
  }
}

// 指令使用
app.directive('permission', {
  mounted(el, binding) {
    const { hasPermission } = usePermission()
    if (!hasPermission(binding.value)) {
      el.parentNode?.removeChild(el)
    }
  }
})

// 使用示例
<el-button v-permission="'user:delete'">删除</el-button>
```

  data: any[],
  columns: Column[],
  filename: string

  const XLSX = await import('xlsx')
  const worksheet = XLSX.utils.json_to_sheet(data)
  const workbook = XLSX.utils.book_new()
  XLSX.utils.book_append_sheet(workbook, worksheet, 'Sheet1')

}

export async function exportToCSV(
  data: any[],
  filename: string
) {
  // 导出CSV
  const csv = data.map(row => Object.values(row).join(',')).join('\n')
  const blob = new Blob([csv], { type: 'text/csv' })
  downloadBlob(blob, `${filename}.csv`)
}
