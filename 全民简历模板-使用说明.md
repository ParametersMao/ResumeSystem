# 全民简历模板 - 使用说明

## 📁 文件说明

已为您生成了两个文件：

1. **`全民简历-标准模板.json`** - 模板配置文件
2. **`全民简历-示例数据.json`** - 示例简历数据

---

## ✨ 模板特点

这是一份**专业的单栏简历模板**，完全按照您提供的图片设计，包含以下特点：

### 视觉特色
- ✅ 顶部带照片的个人信息区（照片在右侧）
- ✅ 蓝色主题色（#2c5aa0）
- ✅ 清晰的模块分隔（蓝色下划线）
- ✅ 专业的排版布局
- ✅ 技能进度条展示

### 包含模块
1. **个人信息** - 带照片、求职意向、基本信息、联系方式
2. **教育背景** - 学校、专业、GPA、主修课程
3. **工作经验** - 公司、职位、时间、工作内容
4. **技能特长** - 技能进度条展示
5. **荣誉证书** - 获奖列表
6. **自我评价** - 个人评价段落

## 🚀 快速使用

### 方法一：通过系统上传（推荐）

1. **登录系统**
   - 打开简历编辑系统
   - 进入模板管理页面

2. **上传模板**
   - 点击"上传模板"
   - 选择 `全民简历-标准模板.json`
   - 等待上传成功

3. **使用模板**
   - 在模板库中选择此模板
   - 创建新简历
   - 填写您的个人信息

### 方法二：本地测试

1. **启动项目**
   ```bash
   cd fronted-resume-web
   npm run dev
   ```

2. **使用诊断工具验证**（推荐）
   - 打开浏览器
   - 找到右下角的"🔍 模板诊断工具"
   - 上传 `全民简历-标准模板.json`
   - 点击"开始诊断"
   - 查看是否有问题

3. **在编辑器中测试**
   - 将模板文件放在合适位置
   - 通过编辑器加载模板
   - 填写内容预览效果

## 📝 自定义修改

### 修改颜色主题

在模板JSON中找到 `styles.colors`：

```json
"colors": {
  "primary": "#2c5aa0",        // 主色（标题下划线、进度条）
  "text": "#333333",           // 主文字颜色
  "textSecondary": "#666666",  // 次要文字
  "textMuted": "#999999",      // 辅助文字（日期等）
  "border": "#e5e7eb"          // 边框颜色
}
```

**示例**：改为绿色主题
```json
"primary": "#27ae60"  // 改为绿色
```

### 调整模块顺序

修改每个模块的 `order` 字段：

```json
{
  "type": "education",
  "order": 1  // 数字越小越靠前
}
```

**示例排序**：
- 0: 个人信息（必须在最前）
- 1: 教育背景
- 2: 工作经验
- 3: 技能特长
- 4: 荣誉证书
- 5: 自我评价

### 隐藏某个模块

将 `visible` 设置为 `false`：

```json
{
  "type": "awards",
  "visible": false  // 隐藏荣誉证书模块
}
```

### 调整个人信息显示字段

在 `personal` 模块的 `config.fields.visible` 中修改：

```json
"visible": [
  "name",
  "age",
  "gender",
  "location",
  "yearsOfExperience",
  "phone",
  "email"
  // 添加或删除字段
]
```

### 修改字体大小

在各模块的 `config` 中找到对应的样式：

```json
"nameStyle": {
  "fontSize": "28px"  // 姓名字体大小
},
"titleStyle": {
  "fontSize": "18px"  // 模块标题字体大小
}
```

## 🎨 样式说明

### 个人信息区域样式

```json
{
  "nameStyle": {
    "fontSize": "28px",        // 姓名字体大小
    "fontWeight": "700",       // 字体粗细
    "letterSpacing": "2px"     // 字间距
  },
  "avatarImageStyle": {
    "width": "100px",          // 照片宽度
    "height": "120px"          // 照片高度
  }
}
```

### 模块标题样式

```json
{
  "titleStyle": {
    "fontSize": "18px",
    "fontWeight": "700",
    "borderBottom": "2px solid #2c5aa0"  // 蓝色下划线
  }
}
```

### 技能进度条样式

```json
{
  "skillBars": {
    "barStyle": {
      "height": "10px",              // 进度条高度
      "backgroundColor": "#e5e7eb"   // 背景色
    },
    "progressStyle": {
      "backgroundColor": "#2c5aa0"   // 进度条颜色
    }
  }
}
```

## ⚠️ 注意事项

### 1. 照片设置
- 照片位置固定在右上角
- 推荐尺寸：100px × 120px
- 支持格式：JPG、PNG
- 如无照片会显示占位符

### 2. 求职意向
- 求职意向会显示在姓名下方
- 包含：期望岗位、薪资、城市、工作时间等
- 可以根据需要添加或删除字段

### 3. 技能进度条
- level 值范围：1-5
- 自动转换为百分比显示
- 可以按类别分组展示

### 4. 文本格式
- 工作职责支持数组形式
- 自动生成项目符号
- 支持HTML格式（谨慎使用）

### 5. 日期格式
- 推荐格式：`YYYY-MM`（如：2020-01）
- 当前工作使用：`至今`

## 🎯 常见问题

### Q1: 照片不显示怎么办？
**A**: 检查：
1. 照片URL是否正确
2. 照片是否可访问
3. 是否设置了 `showAvatar: true`

### Q2: 如何隐藏照片？
**A**: 在 personal 模块中设置：
```json
"showAvatar": false
```

### Q3: 模块标题下划线颜色如何改？
**A**: 修改各模块的 titleStyle：
```json
"borderBottom": "2px solid #YOUR_COLOR"
```

### Q4: 技能进度条颜色如何改？
**A**: 在 skills 模块中修改：
```json
"progressStyle": {
  "backgroundColor": "#YOUR_COLOR"
}
```

### Q5: 模板在系统中显示不正常？
**A**: 
1. 使用诊断工具检查
2. 查看浏览器控制台错误
3. 参考 `模板问题修复指南.md`
