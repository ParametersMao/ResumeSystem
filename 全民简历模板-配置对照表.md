## 1️⃣ 个人信息区域

### 视觉效果（图片中的样子）
```
┌─────────────────────────────────────────────────────────┐
│  全民简历                                    [照片]      │
│  期望岗位：xxx | 期望薪资：xxx | 期望城市：xxx | ...    │
│  29岁 | 男 | 上海 | 7年经验                              │
│  15588888888    qmjianli@qq.com                         │
└─────────────────────────────────────────────────────────┘
```

### 对应配置
```json
{
  "type": "personal",
  "config": {
    "showAvatar": true,           // ← 显示右上角照片
    "avatarPosition": "right",    // ← 照片在右侧
    "nameStyle": {
      "fontSize": "28px",         // ← "全民简历" 字体大小
      "fontWeight": "700"         // ← 字体粗细（加粗）
    },
    "intentionStyle": {           // ← 求职意向那一行的样式
      "fontSize": "14px",
      "display": "flex",
      "gap": "8px"                // ← 字段之间的间距
    },
    "basicInfoStyle": {           // ← "29岁 | 男 | ..." 这一行
      "fontSize": "14px",
      "display": "flex",
      "gap": "8px"
    },
    "contactStyle": {             // ← 联系方式那一行
      "fontSize": "14px",
      "gap": "20px"
    }
  }
}
```

### 数据填充
```json
{
  "profile": {
    "basic": {
      "name": "全民简历",                    // → 大标题
      "age": 29,                            // → 29岁
      "gender": "男",                       // → 男
      "location": "上海",                   // → 上海
      "yearsOfExperience": "7年经验",       // → 7年经验
      "phone": "15588888888",               // → 手机号
      "email": "qmjianli@qq.com",          // → 邮箱
      "intention": {
        "expectedPosition": "市场营销",     // → 期望岗位
        "expectedSalary": "面议",          // → 期望薪资
        "expectedCity": "上海/北京"         // → 期望城市
      }
    }
  }
}
```

## 4️⃣ 技能特长

### 视觉效果
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
技能特长  ← 蓝色下划线
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

语言能力:
普通话     ████████████████████ 熟练
           ← 蓝色进度条

计算机:
计算机     ████████████████░░░░ 良好
           ← 80% 填充

设计:
设计       ████████████░░░░░░░░ 一般
           ← 60% 填充
```

### 对应配置
```json
{
  "type": "skills",
  "config": {
    "layout": "bars",                        // ← 使用进度条布局
    "skillBars": {
      "enabled": true,                       // ← 启用进度条
      "barStyle": {
        "height": "10px",                    // ← 进度条高度
        "backgroundColor": "#e5e7eb"         // ← 背景色（未填充部分）
      },
      "progressStyle": {
        "backgroundColor": "#2c5aa0"         // ← 进度条颜色（蓝色）
      },
      "labelStyle": {
        "fontSize": "14px",                  // ← 技能名称字体
        "fontWeight": "500"                  // ← 稍微加粗
      }
    }
  }
}
```

### 数据填充
```json
{
  "items": [
    {
      "category": "语言能力",                // → 分类名称
      "skills": [
        {
          "name": "普通话",                  // → 技能名称
          "level": 5,                       // → 等级（1-5）
          "levelText": "熟练"               // → 等级文本
        }
      ]
    },
    {
      "category": "计算机",
      "skills": [
        {
          "name": "计算机",
          "level": 4,                       // → 4/5 = 80%
          "levelText": "良好"
        }
      ]
    }
  ]
}
```

**level 与填充率对应**：
- `level: 5` → 100% 填充（精通）
- `level: 4` → 80% 填充（熟练）
- `level: 3` → 60% 填充（良好）
- `level: 2` → 40% 填充（一般）
- `level: 1` → 20% 填充（了解）

## 🎨 整体样式配置

### 颜色方案
```json
{
  "styles": {
    "colors": {
      "primary": "#2c5aa0",        // ← 主色调（蓝色）
                                   //   用于：标题下划线、进度条、圆点
      
      "text": "#333333",           // ← 主文字（黑色）
                                   //   用于：姓名、标题、重要内容
      
      "textSecondary": "#666666",  // ← 次要文字（深灰）
                                   //   用于：正文内容
      
      "textMuted": "#999999",      // ← 辅助文字（浅灰）
                                   //   用于：日期、次要信息
      
      "border": "#e5e7eb",         // ← 边框色
                                   //   用于：分隔线、照片边框
      
      "background": "#ffffff"      // ← 背景色（白色）
    }
  }
}
```

### 字体配置
```json
{
  "fonts": {
    "heading": "Microsoft YaHei, ...",  // ← 标题字体
    "body": "Microsoft YaHei, ..."      // ← 正文字体
  }
}
```

### 间距配置
```json
{
  "spacing": {
    "sectionMargin": "20px",      // ← 模块之间的间距
    "elementMargin": "12px"       // ← 元素之间的间距
  }
}
```

### 页面配置
```json
{
  "globalConfig": {
    "maxWidth": "900px",          // ← 内容最大宽度
    "padding": "40px 50px",       // ← 页面内边距
    "layout": "single-column"     // ← 单栏布局
  }
}
```

## 🔧 快速定制指南

### 想改变主题色？
找到 `styles.colors.primary` 改为您想要的颜色：
```json
"primary": "#27ae60"  // 绿色
"primary": "#e74c3c"  // 红色
"primary": "#9b59b6"  // 紫色
```

### 想调整照片大小？
找到 `personal.config.avatarImageStyle`：
```json
"width": "120px",   // 改为 120px
"height": "150px"   // 改为 150px
```

### 想让姓名更大？
找到 `personal.config.nameStyle.fontSize`：
```json
"fontSize": "32px"  // 从 28px 改为 32px
```

### 想改变进度条颜色？
找到 `skills.config.skillBars.progressStyle`：
```json
"backgroundColor": "#27ae60"  // 改为绿色
```
