这是一份**完全按照您的图片设计**的简历模板，包含：

- ✅ 顶部带照片（右上角，100×120px）

## 🚀 快速开始

### 1️⃣ 验证模板（推荐）

在使用前，建议先验证模板：

```bash
# 启动项目
cd fronted-resume-web
npm run dev
```

然后：
1. 打开浏览器
2. 找到右下角的 **🔍 模板诊断工具**
3. 上传 `全民简历-标准模板.json`
4. 点击 **"开始诊断"**
5. 查看结果：
   - ✅ **有效** → 可以直接使用
   - ⚠️ **警告** → 建议修复
   - ❌ **错误** → 必须修复

### 2️⃣ 上传模板

通过系统上传：
1. 登录系统
2. 进入模板管理
3. 上传 `全民简历-标准模板.json`
4. 设置模板名称和预览图
5. 保存

### 3️⃣ 使用模板

1. 创建新简历
2. 选择"全民简历-标准模板"
3. 填写个人信息
4. 预览效果
5. 导出PDF

## 🔧 快速定制

### 改变主题色
```json
// 在模板JSON中找到：
"primary": "#2c5aa0"

// 改为其他颜色：
"primary": "#27ae60"  // 绿色
"primary": "#e74c3c"  // 红色
"primary": "#9b59b6"  // 紫色
```

### 调整照片大小
```json
// 在 personal.config.avatarImageStyle 中：
"width": "120px",   // 宽度
"height": "150px"   // 高度
```

### 隐藏某个模块
```json
// 将对应模块的 visible 改为 false：
{
  "type": "awards",
  "visible": false  // 隐藏荣誉证书
}
```

### 改变模块顺序
```json
// 修改 order 数字（越小越靠前）：
{
  "type": "education",
  "order": 1  // 第一个（个人信息是0）
}
```

### 🔍 常见问题

**Q: 照片不显示？**
A: 检查 `showAvatar: true` 和照片URL

**Q: 颜色不对？**
A: 检查 `styles.colors.primary`

**Q: 模块顺序不对？**
A: 检查每个模块的 `order` 字段

**Q: 技能进度条不显示？**
A: 确保 `layout: "bars"` 和 `skillBars.enabled: true`
