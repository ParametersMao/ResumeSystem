完全自定义样式功能测试

1. 自定义CSS注入测试

测试模板数据中的 customCss 字段是否能正确注入到页面中。

// 在模板数据中添加： "styles": { "customCss": ` .my-custom-class { background: linear-gradient(45deg, #ff6b6b, #4ecdc4); color: white; padding: 20px; border-radius: 10px; } ` }

2. Section自定义样式测试

测试模块的 customClass 和 customStyle 是否生效。

// 在section配置中添加： "config": { "customClass": "my-custom-section", "customStyle": { "background": "linear-gradient(135deg, #667eea 0%, #764ba2 100%)", "color": "white", "padding": "30px", "borderRadius": "15px" } }

3. JSON Schema渲染测试

测试 schema 字段是否能正确渲染为HTML。

// 在section配置中添加： "config": { "schema": { "type": "flex", "direction": "row", "justify": "space-between", "items": [ { "type": "text", "content": "Hello World", "style": { "fontSize": "24px", "color": "#333" } } ] } }

4. 样式Token系统测试

测试 tokens 字段是否能生成CSS变量。

// 在模板数据中添加： "styles": { "tokens": { "colors": { "primary": "#3498db", "secondary": "#2ecc71" }, "spacing": { "md": "16px", "lg": "24px" } } }

5. 响应式样式测试

测试 responsive 字段是否能生成媒体查询。

// 在模板数据中添加： "responsive": { "breakpoints": { "mobile": "0px", "tablet": "768px", "desktop": "1024px" }, "mobile": { ".my-class": { "fontSize": "14px" } }, "desktop": { ".my-class": { "fontSize": "18px" } } }

测试步骤

  1. 将测试模板 test-template.json 导入到系统中
  2. 在编辑器中查看预览效果
  3. 检查浏览器开发者工具中的CSS是否正确注入
  4. 调整浏览器窗口大小测试响应式效果
  5. 验证Schema渲染的组件是否正确显示

预期效果

  • ✅ 个人展示区应该有渐变背景和白色文字
  • ✅ 技能标签应该有不同颜色，鼠标悬停时有动画效果
  • ✅ 工作经验模块应该有粉色渐变背景
  • ✅ 整体背景应该有渐变效果
  • ✅ 在移动端技能标签应该变小
  • ✅ 所有自定义样式都应该通过CSS变量和媒体查询实现
