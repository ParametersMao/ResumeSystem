## 🔍 关键区别说明

### 模板 vs 简历数据

| 项目 | 模板配置 | 简历数据 |
|-----|---------|---------|
| **文件** | .json 模板文件 | 在编辑器中填写 |
| **内容** | 样式、布局、配置 | 个人信息、经历、技能 |
| **作用** | 定义简历的样式 | 定义简历的内容 |
| **上传位置** | 中台模板管理 | Web 端编辑器 |

### 您的操作流程

```
✅ 第1步：上传模板（已完成）
   ↓
❌ 第2步：填写数据（缺少这一步！）← 这是问题所在
   ↓
✅ 第3步：预览和导出
```

## 🎯 核心要点

### 为什么显示空白？

因为：
1. ✅ 模板已上传（定义了样式）
2. ❌ **数据未填写**（没有内容显示）

就像：
- 模板 = 空白表格
- 数据 = 填入表格的内容

### 正确理解

```
模板 + 数据 = 完整简历

全民简历标准模板.json  （样式配置）
+
您填写的个人信息        （内容数据）
=
最终显示的简历效果      （图2的样子）
```

## 📸 上传照片

### 方法 1: 在编辑器中上传

1. 找到"个人信息"模块
2. 看到"照片"或"头像"上传按钮
3. 点击上传您的照片
4. 照片会显示在右上角（100×120px）

### 方法 2: 使用 URL

如果照片已经在网上：
```
1. 在照片字段输入URL
2. 例如：https://example.com/avatar.jpg
3. 保存后照片会显示
```

### 照片要求

- 推荐尺寸：100×120px 或更大
- 格式：JPG、PNG
- 大小：< 2MB
- 位置：系统会自动放在右上角

## ⚠️ 常见错误

### 错误 1: 只上传模板不填数据
```
❌ 上传模板 → 直接预览 → 显示空白

✅ 上传模板 → 填写数据 → 预览完整简历
```

### 错误 2: 字段名称不匹配
```
❌ 使用模板中不存在的字段

✅ 使用修正版模板中定义的字段
```

### 错误 3: 数据格式错误
```
❌ 技能等级填写文字 "熟练"

✅ 技能等级选择数字 5 或 100%
```
