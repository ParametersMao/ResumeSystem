## 🚀 快速使用

### 1. 启动项目
```bash
cd fronted-resume-web
npm run dev
```

### 2. 打开浏览器
访问: http://localhost:5173

### 3. 找到诊断工具
在页面**右下角**（偏左一点）会看到：

```
┌──────────────┐
│🔍 模板诊断工具 │
│              │
│ [展开]       │
└──────────────┘
```

### 4. 开始诊断

**单个模板诊断**:
1. 点击"展开"
2. 选择"单个模板"标签
3. 上传模板文件或粘贴JSON
4. 点击"开始诊断"
5. 查看问题和修复建议
6. 点击"应用自动修复"
7. 下载修复后的模板

**批量诊断所有模板**:
1. 切换到"批量诊断"标签
2. 点击"开始批量诊断"
3. 查看所有模板的状态
4. 点击"查看详情"了解问题
5. 导出诊断报告
